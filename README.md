
# Image Super Resolution

Проект посвящен повышению качества изображений (Super Resolution) с использованием методов глубокого обучения.

Модель увеличивает разрешение изображения и восстанавливает мелкие детали.

## Функциональность

- Повышение разрешения изображений
- Восстановление деталей изображения
- Улучшение четкости
- Обработка изображений низкого качества

## Используемые технологии

- Python
- PyTorch
- Real-ESRGAN
- Gradio


=

- Установка зависимостей:

